## Getting Started

To run the project locally, follow these steps:

1. **Clone the repository**

   ```bash
   git clone https://github.com/Hasan-Uddin/flutter-bmi-calculator.git
   cd flutter-bmi-calculator
2. **Install dependencies**

   ```bash
   flutter pub get
3. **Run the app**

   ```bash
   flutter run

1. **build**
    - for web
       ```bash
       flutter build web
    - for Android
       ```bash
       flutter build apk
